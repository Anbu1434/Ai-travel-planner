# 🧭 AI Travel Planner App

**Your Personal AI-Powered Travel Assistant**  
Plan smarter, travel better. From trip discovery to itinerary creation — powered by AI.

---

## ✨ Features

- 🔍 **Smart Destination Discovery**  
  Get personalized recommendations based on your interests, budget, and trip duration.

- 🧠 **AI-Generated Itineraries**  
  Automatically create detailed daily plans, optimized for time and convenience.

- 🌦️ **Real-time Weather & Alerts**  
  Stay updated with current conditions and travel advisories.

- 🛫 **Flight & Hotel Suggestions**  
  Compare top travel options within your budget.

- 🗺️ **Interactive Map View**  
  Visualize your entire journey on a sleek map interface.

- 🧳 **Packing Checklist Generator**  
  Get a customized list based on your destination and travel type.

---

## 🖥️ Tech Stack

- **Frontend**: React Native  
- **Backend**: Node.js /Firebase 
- **APIs Used**: Gemini, Google Maps 
- **Database**: Firestore
- **Authentication**: Firebase Auth
